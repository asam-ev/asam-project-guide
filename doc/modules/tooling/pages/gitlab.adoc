GitLab is a web-based DevOps lifecycle tool that provides a xref:git.adoc[Git] repository manager.
It can host wikis, has issue-tracking and continuous integration and deployment pipeline features, using an open-source license.

== GitLab At ASAM
ASAM uses {gitlab-url}[GitLab] for multiple aspects of its standardization process:

* Document control during standard development
* Commenting and tracking of feedback
* Automation of document creation and publication steps
* Milestone tracking
* Synchronizing assets, styleguides, scripts and reusable content
