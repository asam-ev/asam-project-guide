Just some Examples for diagrams to be used in the standards.


== Examples

.PlantUML
[plantuml, osc-architecture-overview, svg]
----
skinparam componentStyle rectangle

database "OSC Model Instance" as OSC
[OSC Director]

OSC --> [OSC Director]: is intepreted
[OSC Director] --> [Simulator Core] : orchestrates the scenarios
----

.BlockDiag
[blockdiag]
----
blockdiag {
orientation = portrait
OS_Model_Instance [shape = flowchart.database, label="OSC Model Instance"];
OSC_Director [label="OSC Director"];
Simulator_Core [label="Simulator Core"];

OS_Model_Instance -> OSC_Director [label = "is intepreted"]
OSC_Director -> Simulator_Core [label = "orchestrates the scenarios"]
}
----
