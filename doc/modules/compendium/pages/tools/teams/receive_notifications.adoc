== Context
Notifications for new posts to Teams channels are not always on by default.
In order to be aware of all updates, the user can turn on the updates.

NOTE: This needs to be done for each channel separately!

== Result
* User receives emails when a notification in ASAM's MS Teams is created for them

== Prerequisites
* User has access to the Teams channel
