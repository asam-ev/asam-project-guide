== Context
Notifications in Microsoft Teams (MS Teams) can be easily overlooked.
In order to be aware of all updates, the user can activate email notifications for the ASAM MS Teams.

== Result
* User receives emails when a notification in ASAM's MS Teams is created for them

== Prerequisites
* User has access to ASAM MS Teams

== How
* Turn these to receive an email when you receive a notification
** We recommend at least personal, channel and team mentions
