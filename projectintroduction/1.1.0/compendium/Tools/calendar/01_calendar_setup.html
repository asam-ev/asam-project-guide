Setting up the OpenX calendar

Context

  • All OpenX projects share a calendar under the email address of openx@asam.net. This MUST be included on all meetings taking place for the projects to make sure it is up to date.

  • The OpenX calendar can be viewed online as well.

Result

You have imported the OpenX calendar in your email program.

Prerequisites

none

How

  1. Copy the ics link to you clipboard

  2. Follow the instructions for your calendar type:

Subscribing with Google

Troubleshooting / Known Issues

I cannot subscribe using the provided link

Make sure you use the ics link, not the web url. Subscribing requires an ics link or file. The web url (link) is ONLY for viewing the calendar online without subscribing to it.
