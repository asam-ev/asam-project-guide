Create an issue for a page

Context

If a page is not correct, the easiest way is to directly write the feedback from the page itself. To do so, you can use the provided "Submit feedback" button.

Result

An issue with the described problem and suggested solution is created.

Prerequisites

None

How

  1. Click Submit feedback

    Figure 1. "Submit feedback" button in Antora project

  2. Edit the title of the form

  3. Select the relevant topics by replacing any - [ ] with - [x] in the "Prerequisites" section

  4. Add a title of the topic to the section "Summary"

  5. If relevant, add the section name on that page to the section "Section"

  6. Add a comprehensive and comprehensible description to the section "Description". Ideally, use screenshots to show what you mean more clearly

  7. Add a suggested solution to the section "Suggestion"

    Figure 2. The issue form for submitting feedback in GitHub

  8. Click Submit new issue
