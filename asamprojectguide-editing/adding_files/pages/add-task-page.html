Adding a Task page

This is a variation on the default page creation. The artifact template is intended for all topics like tasks or step-by-step instructions.

Context

In order to make documented taskss easier to understand, it is necessary to have a standard format for taskss in which they are written. For this purpose, the template-artifact-page.adoc file was created. Each tasks that is documented in this Project Guide shall use the artifact template.

If you want to define a reusable content, consider using a Partial instead. See Pages VS Partials for more information on the difference between those two.

Result

A tasks is created that follows the ASAM artifact template.

Prerequisites

none

How

These instructions already include the requirements of Creating A Standard Page by Antora implicitely. However, if one deviates from this guide here, the Antora restrictions and requirements still apply!

  1. Find the module you want your page to be located in.

    This should be the module which it relates to most closely. See other pages in the modules in question for reference. You can also look at the Project Guide structure for the Project Guide.

  2. Create your file in the "pages" folder or move it there

    1. Alternatively, create a new folder and/or add your file to an existing folder within the "pages" folder if you want to add a structure within the module.

      You can copy the template file from the examples folder in the project-guide module folder.

  3. Add the following standard content to your file and modify it accordingly:

    = Artifact page name
:description: The artefact's description
:keywords: artifact
:page-partial:

include::partial$_attributes.adoc[]
include::{abb}[]

//Remove the backslash in front of the includes before using the page

// If you use a recurring image for similar artifacts, declare it as attribute in partials/_attributes.adoc and reference here.

== Context
Describe the context (why) for this artifact.

== Result
Describe the expected results of this artifact.

== Prerequisites
List all preqrequisits for this artifact.

== How
Describe the steps to perform this artifact.
If you have alternative ways, use subchapters.

== Troubleshooting / Known Issues
// OPTIONAL

If you have steps for this artifact where troubleshooting or known issues exist, add subchapters for each topic.

//OPTIONAL: Add the reference macro with keywords you want to reference.
// Example: related::guide,!main[]   <-- (lists all pages where the keyword "guide" but not "main" were used)
    Remove the backslash in the template file before the include instruction when you copy the template file directly.
