Macro: pages

Description

This macro creates cross-references under the section "Pages" at the macro location to all pages (with or without nested folders) in a given folder relative to the file with the macro. If no folder is defined, the file’s folder is used instead.

In addition to this replacement, an attribute :pagemacro: is automatically set so that the section numbering feature will exclude this section title automatically. This helps other Antora extensions to identify this particular section header in order to act on it differently.

Prerequisites

At least one asciidoc file that is not this file must be in the target folder.

Parameters

Parameter Argument(s) Note

all

-

Also include all sub-folders

path

relative-path-to-folder

Path to the target folder relative to this file

Usage

Add page::[<arguments>] in an empty line and replace <arguments> with a list of arguments, separated by "," (comma). If you add no arguments, the macro will list all other pages in the same folder as this file.
