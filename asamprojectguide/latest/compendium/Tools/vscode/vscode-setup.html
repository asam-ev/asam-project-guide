Setting up Visual Studio Code

Context

Visual Studio Code is the recommended editor for working on AsciiDoc files and diagrams locally. It is also a useful tool for interfacing with git.

Result

The user has installed Visual Studio Code and configured it with the recommended settings.

Prerequisites

none

How

  1. Download Visual Studio Code (VSCode) here

  2. Follow the instructions during installation

  3. After installation, start VSCode for further configuration

For recommended settings and extensions, see the guides on Visual Studio Code.
