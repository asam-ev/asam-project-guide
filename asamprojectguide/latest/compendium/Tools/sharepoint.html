Microsoft Sharepoint

Click the Sharepoint icon on the right to directly access ASAM’s general Sharepoint.

Sharepoint is a Microsoft service that is used to manage files. It can be accessed directly or through Microsoft Teams.

Project files such as chapters, specifications, or images intended for use in the standard are to be handled through GitLab instead!
