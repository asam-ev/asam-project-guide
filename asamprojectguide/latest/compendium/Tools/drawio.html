Draw.io

Click the tool’s icon on the right to access the online tool directly.

Draw.io is a free tool for creating diagrams. The tool can be downloaded and run locally (as Draw.io Desktop), accessed through a web browser (through Draw.io web link with ASAM library), or be used through the Visual Studio Code extension (id: hediet.vscode-drawio).

ASAM recommends using the web app.

It is recommended to use the VS Code extension and Draw.io Desktop only for viewing diagrams and making small alterations, not creating them or making bigger changes. The reason is that the ASAM library cannot be integrated directly as of now into the extension solution.

Draw.io at ASAM

ASAM uses Draw.io for creating custom diagrams for standards and documentation.
