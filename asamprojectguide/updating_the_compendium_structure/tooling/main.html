Tools Overview

Find help on setting up your work environment for ASAM projects.
This part contains setup instructions and guides for all mandatory and recommended tools.

Most roles can contribute by Working Online.

Tools in the Project Guide are separated into mandatory infrastructure and recommended tools.
Mandatory infrastructure contains everything that is mandatory for every project. These platforms and tools are often hosted by ASAM and always accessible online. Find out more about it in Working Online.
Recommended tools are tools that in provide additional features beyond their online equivalent and can be beneficial for certain users. ASAM provides helpful guides and tips here.

It is also a central aspect of ASAM’s concept of Documentation As Code.
