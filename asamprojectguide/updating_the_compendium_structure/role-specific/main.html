ASAM Roles

Get to know your role(s) and responsibilities.
This part contains detailed information on all roles typically found in and related to ASAM projects. Each role also links to relevant How-Tos, installation instructions, and other role-related information.

Click on a role’s label to navigate directly to the role’s summary page.
Table 1. List of roles
Role Parent Role Involved in Description

GTM

nA

All phases

A GTM (Global Technology Manager) is an ASAM employee focussed on coordinating project processes and the background work behind a project.

ASAM Member

nA

  • Ideation phase

  • Proposal phase

  • Review phase

A member of ASAM

Idea Giver

STANDARD USER

  • Ideation phase

  • Proposal phase

The submitter of an idea for a new ASAM project. Must become an ASAM member (if not) before the project starts to continue participation.

Proposal Submitter

nA

Proposal phase

A member of ASAM

WG MEMBER

ASAM Member

  • Proposal phase

  • Development phase

A project member working in a particular workgroup, a subdivision of the overall project. If the project is split up into smaller workgroups, a project member can join a workgroup to become a workgroup member.

PROJECT LEAD

WG MEMBER

Development phase

The person responsible for leading and coordinating all project activities. Without an elected project lead, the project may not start. The project members must elect a project lead at the beginning of the project.

WG LEAD

WG MEMBER

Development phase

If the project is split up into smaller workgroups, a project member can be elected to be the workgroup lead and will support the project lead. Each workgroup should have zero or one workgroup lead.

SERVICE PROVIDER

nA

Development phase

Provides additional services necessary for the standard development. If the project needs additional resources, a service provider can be hired, which requires an approved budget by the TSC.

TECHNICAL WRITER

nA

Development phase

Formats and writes the document(s) of the standard based on the provided input from the WG members in a formalized and standard-conform way. If the project requires a technical writer, either the ASAM technical writer will fill this role or a service provider can be hired to fill that role.

STANDARD USER

nA

An ASAM Standard user is someone that is part of the ASAM community but not involved in the project development at hand.

STANDARD MANAGER

WG MEMBER

The manager at ASAM responsible for the standard.

TSC Member

ASAM Member

  • Proposal phase

  • Review phase

The TSC is responsible for approving or rejecting budgets, projects and proposals as well as releases of project results, such as standards or concept papers.

REVIEWER

STANDARD USER / ASAM Member

Review phase

An ASAM Standard user or WG member that reviews a draft or release candidate for a new or updated standard.
