Starting a container from docker desktop

This works if the container was already created through either using the VSCode method or the Powershell method.

  1. Open Docker Desktop (if you already started it, it may be in the system tray)

  2. Go to "Containers/Apps" and select your container from the list

  3. Click the button to run the container
